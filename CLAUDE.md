# 文学常识题库查证规范

## 查证方法

每题从三个维度独立验证，缺一不可：

1. **题目与选项的匹配性** — 选项内容是否与题目相关、选项之间是否构成合理干扰项、选项名称/人物是否真实存在（不可用假名充数）
2. **正确选项与解析的一致性** — 标记的正确答案是否与解析文字描述一致、解析是否在解释正确答案而非其他选项
3. **正确选项的事实正确性** — 答案在历史和文学事实上是否正确、解析中的引文/出处/年代是否准确、涉及生僻知识点时需检索确认

## 常见错误模式

- **答案错但解析对**：答案标记为 A，解析却正确描述了 C 是答案
- **选项张冠李戴**：选项内容与题目主题无关，纯属拼凑
- **假选项充数**：选项中使用不存在的书名、人名、地名
- **解析答非所问**：解析讲的是另一道题的内容
- **年代/朝代错误**：将作品归入错误的历史时期
- **引文归属错误**：将 A 作者的诗句归于 B 作者

## 题库格式

每道题结构：`{ id, question, options[], answer, explanation }`
- `answer` 为选项字母 A/B/C/D
- `options` 格式：`"A. 选项内容"`
- 改动 options 或 answer 后必须同步修正 explanation

## 数据处理原则

- questions.json 为单一数据源，index.html 中的 EMBEDDED_QUESTIONS 需同步更新
- 修改时先用 Edit 工具精确定位，避免整行替换导致格式错乱
- 中文全角引号「」与半角引号"在 Edit 匹配时不可混用
